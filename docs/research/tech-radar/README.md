# Tech Radar

This area stores research and pre-research material for candidate tools, adapters, providers, workers, and references.

It is not canonical architecture. It does not override Book 01, Book 02, or Book 03. It does not define product scope. It does not approve integrations.

Tech radar records candidate tools, risks, license concerns, deployment concerns, POC plans, and decisions so they can be reviewed before any publication or implementation commitment.

Tools do not define Mo. Mo defines the business architecture; tools are evaluated as replaceable support components.
