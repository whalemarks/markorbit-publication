# PORTFOLIO-PLN-0004 — Cross-Book Publication Copyedit

## Purpose

Create a publication-layer style and reference control system for all seven frozen books.

## Sequence

```text
scan frozen books
→ classify safe normalization findings
→ classify ambiguous or semantic findings
→ approve a publication overlay
→ generate normalized release candidates
```

## Gates

- Safe typography, title and unambiguous link fixes may enter a publication overlay.
- Ambiguous references require editorial review.
- Meaning-changing corrections require a separate change proposal.
- Frozen source files remain unchanged in this task.
