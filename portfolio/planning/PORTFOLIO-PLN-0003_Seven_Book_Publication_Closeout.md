# PORTFOLIO-PLN-0003 — Seven-Book Publication Closeout

## Goal

Move the seven frozen editorial baselines from architecture-complete manuscripts to a coherent, publication-ready MarkOrbit book series without reopening accepted architecture.

## Programme sequence

### Gate 1 — Frozen baseline alignment

```text
PUB-TASK-PORTFOLIO-B01-B03-FREEZE-01
```

Freeze Book 01 and Book 03 and confirm seven-book frozen-baseline coverage.

### Gate 2 — Publication readiness audit

```text
PUB-TASK-PORTFOLIO-PUBLICATION-AUDIT-01
```

Audit all seven frozen books for front matter, title consistency, terminology, cross-book references, figures, indexes, source notes, copyright readiness and rendering prerequisites.

### Gate 3 — Common front matter architecture

```text
PUB-TASK-PORTFOLIO-FRONT-MATTER-01
```

Define series title page, copyright page, author/project statement, series map, reading order, version statement and publication disclaimers.

### Gate 4 — Cross-book publication normalization

```text
PUB-TASK-PORTFOLIO-COPYEDIT-01
```

Resolve only publication-level inconsistencies: typography, punctuation, heading conventions, title/subtitle form, glossary labels, reference links and figure captions. Any semantic change requires a separate change proposal.

### Gate 5 — Visual and rendering system

```text
PUB-TASK-PORTFOLIO-RENDER-SPEC-01
```

Specify cover family, page dimensions, typography, heading hierarchy, diagrams, tables, code blocks and output rules for PDF, EPUB and print.

### Gate 6 — Rendered release candidates

```text
PUB-TASK-PORTFOLIO-RENDER-RC1
```

Generate and validate seven rendered release candidates, including link checks, table-of-contents checks, image resolution, page overflow and accessibility checks.

### Gate 7 — Publication and distribution decision

```text
PUB-TASK-PORTFOLIO-RELEASE-DECISION-01
```

Record the explicit Owner decision for public release, commercial distribution and release-channel scope.

## Control rule

Frozen editorial content remains immutable throughout publication preparation. Copyediting that changes meaning, authority, architecture, scope or responsibility must stop and open a separately authorized content change proposal.
