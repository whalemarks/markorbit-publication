# PORTFOLIO-DEC-0001 — Publication Closeout Sequence

## Decision

The seven-book frozen portfolio must pass a common publication closeout programme before any individual book is rendered or released.

## Authorized sequence

```text
1. Seven-book publication readiness audit
2. Common series front matter and release metadata
3. Cross-book publication copyedit and reference normalization
4. Visual identity and rendering specification
5. Seven rendered release candidates
6. Final rendered-candidate acceptance
7. Explicit publication and distribution decision
```

## Rationale

Common assets and rules should be fixed once before seven separate production streams begin. Rendering before title, copyright, edition, terminology, reference and series-identity decisions are normalized would create avoidable rework and inconsistent release objects.

## Next task after audit acceptance

```text
PUB-TASK-PORTFOLIO-FRONT-MATTER-01
Common Series Front Matter and Release Metadata
```

This next task may create publication-layer material but may not alter frozen manuscript meaning. Any semantic correction discovered during publication preparation must be routed through the relevant book's change-control process.

## Authority boundary

Publication and commercial distribution remain unauthorized. ISBN acquisition, channel selection, pricing and public release require separate Owner decisions.
