# PORTFOLIO-REV-0002 — Cross-Book Copyedit Control Review

## Review conclusion

The seven-book portfolio requires one shared publication style sheet and a machine-readable normalization layer before rendering. This task establishes that control layer without altering frozen prose.

## Accepted controls

- seven canonical titles;
- defined architecture-term capitalization;
- protected semantic distinctions;
- rendering-safe punctuation and spacing rules;
- reference and link classification;
- explicit separation between safe copyedit and semantic change.

## Current gate

The control layer may be merged after validation. Applying findings to release candidates remains a later task.
