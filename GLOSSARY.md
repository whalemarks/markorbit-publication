# Glossary

## Publication Repository

The unified repository that manages MarkOrbit book publication structure, governance, shared editorial assets, reviews, releases, and Codex task records.

## Book

A MarkOrbit publication with its own manuscript, book-specific specifications, editorial files, and release history.

## Manifest

A registry file that identifies repository purpose, planned books, canonical paths, and publication status.

## Governance

The publication-level rules that define repository ownership, book ownership, Codex boundaries, and architecture change control.

## Review Round

A defined editorial or publication review cycle with an explicit book, scope, reviewer role, and outcome.

## Release Candidate

A candidate publication package prepared for final review before publication.

## Codex Task

An explicit repository task assigned to Codex for implementation, reporting, or maintenance within approved boundaries.
