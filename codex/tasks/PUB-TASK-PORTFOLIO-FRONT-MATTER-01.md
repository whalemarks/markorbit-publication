# PUB-TASK-PORTFOLIO-FRONT-MATTER-01

## Title

Common Series Front Matter and Release Metadata

## Objective

Create a repository-controlled, non-destructive front-matter package for all seven frozen MarkOrbit books.

## Required outputs

- one common series page;
- one common MarkOrbit project introduction;
- seven controlled book front-matter objects;
- one portfolio release-metadata registry;
- one Owner decision register for unresolved legal and distribution fields;
- validation and GitHub Actions workflow.

## Constraints

- Do not modify frozen manuscript prose or freeze manifests.
- Do not invent an author legal name, copyright owner, publisher, ISBN, publication place or distribution authorization.
- Represent unresolved fields explicitly as `OWNER DECISION REQUIRED`.
- Use the accepted English titles in the current portfolio registry.
- Publication, ISBN procurement and distribution remain unauthorized.

## Acceptance criteria

```text
Books covered: 7 / 7
Book front-matter objects: 7 / 7
Common series page: PRESENT
Project introduction: PRESENT
Release metadata records: 7 / 7
Unresolved legal fields explicitly gated: PASS
Frozen manuscript modifications: 0
Publication authorization: NOT GRANTED
```
