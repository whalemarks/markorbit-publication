The reconstruction follows this authority order:

```text
accepted Core baseline

→ reconciled Book 02 v2 manuscript
→ other v2 manuscripts

```

The manuscript may explain, organize and clarify the frozen baseline. It may identify candidate shared semantics. It does not silently activate new Core objects, lifecycles, authority or migration obligations.
