# Book 04 Source Integration Register

## Purpose

This register prevents the publication reconstruction from becoming a clean-slate rewrite that loses accepted distinctions or useful material.

## Content disposition model

Every inherited topic is classified as:

```text
retain — meaning remains valid and should survive
expand — valid but too compressed
merge — duplicated across several short sections
relocate — belongs in another chapter or Book
qualify — needs clearer boundary or exception
retire — no longer consistent with the accepted architecture
```

## Open editorial questions

1. Whether Personal Workplace is a distinct semantic subtype or an Organization-backed operating profile.
2. How representation changes when a professional acts for several Organizations on the same day.
3. Whether Engagement should be introduced as candidate terminology or expressed through active contracts until Core review.
4. How much of Product Installation belongs in Book 04 versus Product-specific Books.
5. Which Workplace-health indicators are descriptive and which may influence routing or authorization.

These questions remain explicit; prose must not convert them into silent decisions.
