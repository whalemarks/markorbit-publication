# Next Controlled Work Packages

## WP-01 — Frozen Core work-object audit

Compare existing Task, Workflow, Actor, Evidence, Review, Matter, Order, Handoff and Return contracts against:

```text
Work Package
Assignment
Contribution
Outcome
```

Output:

- field-by-field equivalence matrix;
- semantic gaps;
- lifecycle gaps;
- authority owner;
- migration impact;
- recommendation: compose, profile or Change Proposal.

This is the highest-priority next task.
