## Scope Summary

Book 04 defines Workplace as an independent organizational Orbit; organization identity and context; people and accountability; private Knowledge and authorized AI Context; operating surfaces; data boundaries; consumption of Information, Knowledge, Intelligence, Capability and Skill; AI placement; recommendations; handoff to governed Execution; Product Architecture principles; Product embedding and cross-Product handoffs; architectural profiles for Lite, MarkReg and MGSN Gateway; Asset, Content, Artifact, Document, Render, Edit, Delivery, Publish, formalization, outcome and feedback boundaries; network participation; Routing; Capability Evidence; Trust; and ecosystem learning without centralization.

## Non-Goal Summary

Book 04 is not a replacement for Book 01, a semantic amendment to Book 02, a replacement for Book 03, a Product feature catalog, a Lite PRD, a MarkReg PRD, an MGSN marketplace specification, an Agent framework tutorial, a Prompt-engineering manual, an Information & Value Stack manual, a microservice decomposition plan, a database schema specification, an API implementation specification, a deployment architecture, a code-repository design, or a production-readiness authorization.

Book 04 must not redefine Core Objects or Services; redefine Workflow, Task, Human Review, or Event ownership; authorize autonomous AI professional action; authorize external Communication send, filing, submission, payment, provider instruction, or official recordal; treat Distillery candidates as approved Knowledge or Capability; treat recommendations as formal Tasks or Opportunities; treat Artifact as identical to Document; treat MGSN as an open bidding marketplace; or treat Workplace as a centrally owned platform account.
