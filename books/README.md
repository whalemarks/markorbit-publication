## Baseline Rule

Books 01–04 form the pre-Product Portfolio Baseline.

Book 02 remains frozen. Semantic changes require the existing Change Proposal process.

Books 05–07 must consume the baseline and may not silently redefine it.

Book 05 RC1 is frozen through its immutable content baseline, permanent release record and `release/book-05-rc1` pointer.

Book 06 has accepted Product Charter v0.3, Product Baseline v0.1, Chapter Map B06-TOC-V0.1 and Waves 1–6. The current Wave 7 candidate drafts CH30–CH33 as Complete Draft 1 and completes all 34 planned chapter files on owner merge.

Wave 7 merge authorizes a separate whole-book Complete Draft 1 review. It does not itself accept the whole book as Complete Draft 1 or authorize Release Candidate status, implementation, production or distribution.
