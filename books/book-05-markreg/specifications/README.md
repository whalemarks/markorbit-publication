## Controlled Record Classes

B05-SPEC-0001 v0.2 uses:

```text
MR-A — Product artifact
MR-C — scoped context
MR-D — accountable decision
MR-E — evidence or source-backed record
```

The class indicates Product meaning. It does not grant authority.
