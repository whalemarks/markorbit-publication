# 00 Preface

This is a scaffold for the Book 03 preface, not a full chapter.

## Purpose

Book 03 explains how MarkOrbit turns Core contracts into governed operational execution.

## Audience

The intended audience includes authors, reviewers, architects, operators, and implementation planners who need a shared execution-system vocabulary before product-system work begins.

## Relation to Book 02 and Book 04

Book 03 consumes Book 02 Core contracts and translates them into execution coordination concepts. It prepares the ground for Book 04 Product System without defining product UI or treating Workplace as the subject of this book.
