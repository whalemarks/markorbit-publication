## Product Identity

MGSN is the platform-owned and platform-governed global professional service capability network through which demand-side users obtain services beyond their own direct coverage.

```text
Platform-governed supply
+ aggregated demand
+ unified procurement
+ recommended best-route matching
+ user final confirmation
+ controlled funds and fulfillment
= reliable global capability
```

## Network Topology

```text
Demand-side Workplace / Lite
            ↓
          MGSN
            ↑
Provider Workplace / Provider Interface
```

MGSN does not create an independent peer-to-peer participant network.

## Delivery Routes

```text
R1 — External Self-Managed Route
R2 — MGSN Recommended Managed Route
R3 — MGSN Managed Preferred-Provider Route
```

R1 remains outside platform procurement, funds and fulfillment guarantees and returns through manually supplied Evidence.

R2 is the default managed route. MGSN may automatically match and preselect the Recommended Best Route, while ordinary transactions retain user final confirmation.

R3 permits user preference subject to platform admission, qualification, conflict, availability, procurement, service and risk controls.

## Initial Commercial Model

```text
Platform wholesale procurement
+ managed transaction service
+ optional subscription / Product entitlement
```

The platform controls provider procurement terms and may offer wholesale, fixed-package or recommended customer-facing pricing according to the service model.

## Funds Direction

MGSN-managed routes require platform-controlled funds rules, evidence-based release, settlement and reconciliation.

Implementation must use a legally suitable platform entity or regulated partner. The Charter does not authorize unlicensed funds custody.
