## Professional Work Products

```text
Purpose and Authorized Context
→ Structured Content
→ Artifact Draft
→ immutable Artifact Version
→ Review Package
→ Render / Edit
→ Readiness Result
→ Delivery / Communication / Publish Package
→ final confirmation and external operation
→ typed Outcome
→ Feedback / Correction
→ Reuse / Retirement
```

Core distinctions:

- Content is not Artifact;
- Artifact is not Document, Evidence or file;
- raw AI text is not an approved professional result;
- later substantive edits create a new version;
- Review and user confirmation are distinct;
- Render complete is not approved;
- Readiness is not external completion;
- Publish Package ready is not published;
- unknown outcomes remain unknown;
- reuse requires purpose, rights, source and Review continuity.
